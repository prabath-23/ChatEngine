How to Build Tools
1
Click "Add Tool" in the Tools tab
Go to the Tools tab and click Add Tool. You'll see a form builder with two tabs: Form Builder (visual) and JSON Schema (advanced).
2
Name & describe your tool
The name is what the AI calls (use snake_case, no spaces). The description tells the AI when to use it — write it clearly!
Name: send_reminder Description: Set a timed reminder that shows as a browser notification after N minutes
3
Add parameters
Parameters are the inputs the AI fills in. Add each one with a name, type, and description. The AI reads the description to know what to pass.
message string "The reminder text" minutes integer "Delay in minutes"
4
Write the JavaScript implementation
Your code receives args (an object) and must return a string. This string is sent back to the AI as the tool result.
const { message, minutes } = args; setTimeout(() => { new Notification('Reminder', { body: message }); }, minutes * 60000); return `Reminder set for ${minutes} min: "${message}"`;
Tip: You can open new tabs, download files, show notifications, or call any browser API — it all runs in your browser.
5
Use it in chat
Just ask naturally. The AI decides when to call your tool:
"Remind me to check my email in 30 minutes" "Generate a portfolio for John Doe, frontend dev" "Download this code as a Python file"
Best model for tools: Use Nemotron 70B Instruct ★ — it follows tool instructions most reliably.
6
Advanced: JSON Schema tab
For complex tools, switch to the JSON Schema tab and write the full OpenAI-compatible tool schema. Supports nested objects, enums, arrays.
{ "name": "create_task", "description": "Create a task", "parameters": { "type": "object", "properties": { "title": { "type": "string" }, "priority": { "type": "string", "enum": ["low","medium","high"] } }, "required": ["title"] } }
